# Project Context

## What This Repository Is

- Repository: `agam-lang/benchmarks`
- Purpose: Provide a standalone, reproducible cross-language benchmark suite for comparing Agam's performance against C, C++, Rust, Go, Python, and other languages.
- Technology: Agam + comparison language sources + Python harness

## Organization Context

- This is part of the `agam-lang` ecosystem.
- The core Agam compiler lives at `agam-lang/agam`.
- The standard library lives at `agam-lang/std`.
- The scientific platform lives at `agam-lang/agamlab`.

## Key Areas

- Algorithm Benchmarks
- Numerical Benchmarks
- Memory Benchmarks
- Compile-Time Benchmarks
